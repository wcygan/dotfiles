---
description: Research and synthesize information from URLs and web searches
---

Read, analyze, and synthesize information from web sources into clear, actionable summaries.

**Usage modes:**

1. **URL synthesis**: Provide URLs to read and summarize
2. **Topic research**: Provide a topic to search and research
3. **Comparative analysis**: Compare information across multiple sources

**Process:**

1. **Gather**: Fetch content from provided URLs or search for topic
2. **Extract**: Identify key information, facts, and insights
3. **Synthesize**: Combine information into coherent narrative
4. **Structure**: Organize by themes or importance
5. **Cite**: Reference sources for each claim

**Output format:**

## Summary
High-level overview (2-3 sentences)

## Key Findings
- Finding 1 [source]
- Finding 2 [source]
- Finding 3 [source]

## Detailed Analysis
Organized by theme or chronology, with source citations

## Sources
1. [URL] - Brief description
2. [URL] - Brief description

**Quality standards:**
- Accuracy: Only include information from sources
- Clarity: Use plain language, define technical terms
- Completeness: Cover main points from all sources
- Objectivity: Present multiple perspectives when present
- Citations: Every claim traceable to source

**Special modes:**

*Quick mode*: Just provide bullet points of key facts
*Deep mode*: Include methodology, limitations, contradictions
*Comparative mode*: Highlight agreements/disagreements across sources
