# Aesthetic Directions

Pick **one** per page. State it explicitly before writing JSX. Every downstream choice (type, color, motion, layout) follows from the commitment.

This file is a **library**, not a menu. Do not default to the first three. Use [domain-matching](./domain-matching.md) to narrow to a pool of 4–6 candidates appropriate to the product's domain, then pick from there.

## 1. Editorial

- **Vibe**: magazine, longform journalism, intentional typography
- **Type**: serif display (Fraunces, Canela, Tiempos) + sans body (Inter Tight, Söhne)
- **Color**: warm off-white + deep ink + one accent (ochre, oxblood, forest)
- **Motion**: staged hero, slow fades, scroll-reveal
- **Layout**: asymmetric, hanging drop-caps, generous leading
- **Ref**: nytimes.com/interactive/*, itsnicethat.com, rauno.me

## 2. Swiss / Modernist

- **Vibe**: grid, restraint, perfect alignment, minimal decoration
- **Type**: geometric sans (Söhne, Neue Haas Grotesk) as display *and* body; switch by weight and size
- **Color**: monochrome + one strong accent (red or yellow)
- **Motion**: minimal; nothing if possible; sharp cuts over fades
- **Layout**: strict 12-column grid, abundant whitespace
- **Ref**: linear.app, vercel.com (pre-2024)

## 3. Brutalist

- **Vibe**: raw, honest, breaks conventions, celebrates the material
- **Type**: heavy sans display (Neue Haas Grotesk Display Black, Druk Wide) + mono body
- **Color**: harsh contrast, unapologetic yellows/cyans/reds on black
- **Motion**: sharp cuts, instant state changes, no easing
- **Layout**: grid violations, oversized type, exposed borders, visible structure
- **Ref**: bloomberg.com/businessweek (some stories), nothing.tech

## 4. Dark / Moody

- **Vibe**: cinema, game UI, night-time tools
- **Type**: serif accent (Reckless Neue) + sans body (Söhne Breit) + mono for numbers
- **Color**: near-black base (oklch(15% ...)), single cyan/violet/ember accent, glowing hover states
- **Motion**: slow and cinematic — 800–1200ms orchestrated entrances
- **Layout**: spotlit content, heavy vignetting, radial gradients
- **Ref**: linear.app (dark), arc.net, superhuman.com

## 5. Handcrafted / Warm

- **Vibe**: made by a person, small studio, not a tech company
- **Type**: classical serif (Canela, Tiempos) + humanist sans (Söhne, Reckless)
- **Color**: warm neutrals (cream, stone, rust) + dusky accent (plum, olive, terracotta)
- **Motion**: gentle, organic, spring-based
- **Layout**: asymmetric, textured backgrounds (subtle grain), imperfect alignment
- **Ref**: ghost.org, readwise.io, frankchimero.com

## 6. Lo-fi Zine / Retro

- **Vibe**: photocopied zine, 8-bit nostalgia, deliberately ugly-beautiful
- **Type**: pixel or typewriter display (VT323, Departure Mono, PP Mondwest) + clean body
- **Color**: high contrast, flat — black + one neon + one pastel
- **Motion**: step-animations (no easing), glitch cuts, scan-lines
- **Layout**: collage, overlapping elements, rotation, stickers
- **Ref**: are.na, bratprincess.xyz, newer indie portfolios

## 7. Serif-forward Tech

- **Vibe**: editorial-meets-product, the Stripe-adjacent bucket
- **Type**: large serif display (GT Super, Söhne Breit) + geometric sans body (Söhne)
- **Color**: pale off-white base + deep navy/forest + one vivid accent
- **Motion**: measured page-load cascades, scroll-reveal
- **Layout**: strong horizontal rhythm, full-bleed imagery interspersed with contained text
- **Ref**: stripe.com, vercel.com (2024+), coinbase.com/institutional

## 8. Neo-brutalist / Memphis

- **Vibe**: anti-polish, playful-aggressive, cardboard-box construction
- **Type**: heavy sans (Space Grotesk is OK here, specifically — or Neue Machina, Inter Display) + mono for tags
- **Color**: primary-adjacent (hot pink, electric cyan, lemon yellow, pure black) with white gaps
- **Motion**: instant state changes, chunky shadows shift on hover, no easing
- **Layout**: offset shadows on cards, thick black borders, grid-but-wrong, tape/sticker decoration
- **Ref**: gumroad.com, fey.com, framer.com (2023 era), neobrutalism.dev

## 9. Cyberpunk / Neon

- **Vibe**: Blade Runner, Shadowrun, hot neon cutting through pollution-dark
- **Type**: wide display (Orbitron, Michroma, PP Neue Machina) + mono body
- **Color**: oklch(10% ...) base + acid neon accents (magenta + cyan, or green-on-black)
- **Motion**: flicker, scan-line overlays, text-glow shimmer, fast reveal
- **Layout**: HUD-like data density, thin hairline borders, corner brackets, CRT curvature
- **Ref**: cyberpunk.net, synthwave-era sites, hyperspace-style game marketing

## 10. Corporate-institutional

- **Vibe**: bank, law firm, consultancy, regulator; "we've been here a hundred years"
- **Type**: classical serif (Tiempos, GT Sectra) + neutral sans body (Söhne, Inter Tight)
- **Color**: ivory/cream + navy or bottle-green + single conservative accent (burgundy, deep gold)
- **Motion**: almost none — dignified fades only, nothing springy
- **Layout**: symmetric, column-based, library-card spacing, generous margins
- **Ref**: jpmorganchase.com (certain pages), economist.com, patagonia.com/stories

## 11. Y2K / Frutiger Aero

- **Vibe**: optimistic tech-utopia, bubble text, aquarium, glossy gels
- **Type**: rounded sans display (Eurostile, Agency FB, Chillax) + clean sans body
- **Color**: aqua blue → lime gradients, chrome highlights, glass, rainbow halos on white
- **Motion**: bounce on hover, ripple clicks, shimmer passes across buttons
- **Layout**: glossy pills and lozenges, water droplets, reflection effects
- **Ref**: early-2000s Mac OS X, Vista aero glass, frutigeraero.com archives

## 12. Glassmorphism / Aurora

- **Vibe**: Apple keynote, frosted-glass overlays, aurora borealis background
- **Type**: precise geometric sans (SF Pro display, Inter Display, Söhne) — medium-heavy display, regular body
- **Color**: deep base + vivid gradient orbs behind blurred glass surfaces
- **Motion**: soft parallax on gradients, blur transitions, lift-on-hover
- **Layout**: floating panels with backdrop-blur, layered depth, edge-lit hairlines
- **Ref**: apple.com/iphone, visionpro.com, many web3 landing pages

## 13. Blueprint / Technical

- **Vibe**: engineering drawing, exploded isometric, schematic honesty
- **Type**: technical mono (JetBrains Mono, IBM Plex Mono) + precise sans (GT America, Söhne)
- **Color**: pale blue paper + dark navy lines, selective red/orange annotation, graph-paper grid
- **Motion**: lines drawing in (SVG stroke-dashoffset), points plotting sequentially
- **Layout**: visible measurement callouts, dimension arrows, isometric illustrations, tight grids
- **Ref**: dyson.com engineering pages, hardware startup sites, rauno.me writeups

## 14. Terminal / CLI

- **Vibe**: monospace all the way down, text-first, "made by engineers"
- **Type**: mono display + mono body (JetBrains Mono, Berkeley Mono, IBM Plex Mono)
- **Color**: terminal green-on-black, or solarized, or a single accent; flat, no gradients
- **Motion**: cursor blink, typewriter reveals, `cat`-style line-by-line entry
- **Layout**: fixed-width columns, ASCII dividers, `$ prompt >` UI, unix-style alignment
- **Ref**: charm.sh, warp.dev, ghostty.org, many DX tools

## 15. Risograph / Print

- **Vibe**: screen-printed poster, limited-palette zine, paper texture
- **Type**: chunky display (PP Editorial New, Ogg, Tiempos Headline) + humanist sans
- **Color**: 2–3 flat inks (fluoro pink + teal + black is canonical), visible paper grain
- **Motion**: subtle — just enough to feel not-static; misaligned print jitter
- **Layout**: posterized compositions, duotone photos, offset registration marks
- **Ref**: risotto-studio.com, independent bookshop sites, indie album art

## 16. Minimalist-luxury / Fashion

- **Vibe**: fashion house, premium watch, art gallery; "if you have to ask"
- **Type**: thin serif display (Didot, Bodoni, Reckless Light) + thin sans body
- **Color**: off-white/cream + one dark (charcoal or cocoa), nothing else
- **Motion**: slow, slow, slow — 1–2s fades, glacial scroll-reveal
- **Layout**: extreme whitespace, tiny type islands in vast empty fields, full-bleed product photography
- **Ref**: apple.com/watch, hermès.com, aesop.com

## 17. Maximalist / Collage

- **Vibe**: teen bedroom wall, mood board, dense and alive
- **Type**: multiple clashing families (a serif + a display sans + a script) — deliberate dissonance
- **Color**: saturated everything; no restraint; competing accents
- **Motion**: constant parallax, multiple motion layers, never-still
- **Layout**: layered images, rotation, tape, washi, overlapping z-stacks
- **Ref**: instrumentstudios.com, heyheyhey.work, wearefuturetaste.com

## 18. Pastoral / Botanical

- **Vibe**: slow-living, farm-to-table, forest bathing, wellness without the wellness-industrial vibes
- **Type**: humanist serif (Canela, Söhne, Reckless) with hand-drawn or botanical accents
- **Color**: sage, clay, moss, bone, soft cream; muted earth tones only
- **Motion**: leaves settling, soft spring curves, nothing abrupt
- **Layout**: hand-drawn botanical illustrations, organic curves, imperfect symmetry
- **Ref**: ourplace.com, casper.com (some eras), apartamento-magazine.com

## How to use this document

1. **Check [domain-matching](./domain-matching.md) first** — narrow to a pool of 4–6 for the product's domain.
2. **Within the pool, pick one direction** that genuinely fits the product's voice (or, for a bake-off, pick 3 that are meaningfully different from each other).
3. **Declare it in the first response**:

> "I'll design this as **Risograph / Print** — PP Editorial New display + Söhne body, fluoro pink + teal + black palette on textured cream, subtle misalignment jitter, posterized duotone photography."

Then build. No wavering, no hedging.

## Rules to prevent library drift

- Never default to #1, #2, and #3 just because they're first. The list is unordered for picking purposes.
- If you find yourself reaching for Editorial / Brutalist / Handcrafted three runs in a row, stop and reread [domain-matching](./domain-matching.md) — you're pattern-matching on prior conversations, not the current product.
- A direction you've never picked before is a feature, not a risk. Bake-offs exist specifically to explore them.
