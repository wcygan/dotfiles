# Nikon Z5 II — Key Specs for Photo Management

## Sensor & Resolution

| Property | Value |
|----------|-------|
| Sensor | 24.5MP FX-format BSI-CMOS |
| Max resolution | 6048 x 4032 px |
| Crop modes | DX (APS-C): 3936 x 2624, 1:1: 4032 x 4032, 16:9: 6048 x 3400 |
| Color depth | 14-bit RAW, 8-bit JPEG |
| Dynamic range | ~14.5 EV (RAW) |

## File Formats

| Format | Extension | Notes |
|--------|-----------|-------|
| JPEG | `.JPG` | Fine/Normal/Basic quality, sRGB or Adobe RGB |
| NEF (RAW) | `.NEF` | 14-bit lossless compressed, ~25-30 MB each |
| HEIF | `.HIF` | 10-bit, smaller than JPEG at equivalent quality |
| Video | `.MOV` | H.265 (HEVC) or H.264, up to 4K/60p |
| Video (RAW) | `.NRW` | N-RAW, 12-bit, very large files |

## File Naming

- Default: `DSC_NNNN.JPG` (sequential, wraps at 9999)
- With Adobe RGB: `_DSC_NNNN.JPG` (leading underscore)
- Can be reset in camera menu
- **Collision risk**: multiple sessions may reuse numbers

## JPEG Quality Settings (in-camera)

| Setting | Approximate file size (24MP) |
|---------|------------------------------|
| Fine | 10-17 MB |
| Normal | 5-8 MB |
| Basic | 2-4 MB |

Your Easter 2026 photos (8-17 MB each) were shot at **Fine** quality.

## Video Specs

| Mode | Resolution | Codec | Typical bitrate |
|------|-----------|-------|-----------------|
| 4K/60p | 3840x2160 | H.265 | ~150 Mbps |
| 4K/30p | 3840x2160 | H.265 | ~100 Mbps |
| 1080/120p | 1920x1080 | H.265 | ~80 Mbps |

Audio: PCM 24-bit/48kHz stereo (uncompressed in MOV container).

## Lens Metadata

The Z5 II records full lens information in EXIF:
- Lens model, focal length, aperture
- Focus distance, VR mode
- Useful for organizing by lens or focal length

## Storage

- Dual SD card slots (UHS-II)
- Slot 1: primary, Slot 2: overflow/backup/RAW+JPEG split
