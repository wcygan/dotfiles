## WebFetch Usage

```
// Fetch live doc page
WebFetch("https://tanstack.com/start/latest/docs/framework/react/server-functions",
  "Extract all createServerFn API options and examples")

// Fetch raw markdown (more reliable)
WebFetch("https://raw.githubusercontent.com/TanStack/router/main/docs/start/framework/react/server-functions.md",
  "Extract the complete API reference and code examples")
```

## Fallback Strategy

1. Try GitHub raw markdown URL first (most reliable)
2. Try doc site URL (may get shell only due to CSR)
3. Search GitHub issues/discussions for recent changes
4. Check CHANGELOG: `https://raw.githubusercontent.com/TanStack/router/main/CHANGELOG.md`
